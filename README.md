# SportsLink

A platform connecting Canadian student amature athletes with university and junior programs. Athletes build profiles with stats and highligh links, coaches dicover and recruit them.

## Tech Stack
| Layer | Tool |
| :---: | :---: |
| Frontend | CSS, HTML, JS |
| Backend | Python Django REST |
| Database | PostgreSQL |
| Container | Docker |
